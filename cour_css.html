Introduction à CSS

CSS (Cascading Style Sheets) est le langage qui donne vie au web. Depuis son introduction en 1996, CSS a révolutionné la façon dont nous concevons les interfaces, permettant une séparation claire entre le contenu (HTML) et la présentation (CSS).

Saviez-vous ? CSS3, la dernière version majeure, a introduit des fonctionnalités avancées comme les animations, les transitions, les flexbox, les grid layouts et les media queries pour le responsive design.

🎨 Les fondamentaux de CSS

Sélecteurs CSS :
.classe { ... } #id { ... } element { ... } [attribut] { ... }
Propriétés courantes : 
/* Texte */
color: #333;
font-family: 'Poppins', sans-serif;
font-size: 16px;
text-align: center;

/* Boîte */
width: 100%;
padding: 20px;
margin: 10px auto;
border: 1px solid #ddd;
border-radius: 8px;

/* Arrière-plan */
background: linear-gradient(to right, #2962ff, #448aff);
background-color: #f8f9fa;

/* Positionnement */
display: flex;
justify-content: center;
align-items: center;
position: relative;

✨ CSS Moderne

Flexbox & Grid :

Les nouveaux systèmes de mise en page qui ont révolutionné le design web :

/* Flexbox */
.container {
  display: flex;
  justify-content: center;
  align-items: center;
  gap: 20px;
}
/* Grid */
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}
Responsive Design :

Les media queries permettent d'adapter le design à différentes tailles d'écran :

/* Mobile first */
.container { padding: 10px; }

/* Tablettes */
@media (min-width: 768px) {
  .container { padding: 20px; }
}

/* Desktop */
@media (min-width: 1024px) {
  .container { padding: 30px; }
}
