Introduction à HTML

HTML (HyperText Markup Language) est le langage fondamental du web. Créé en 1991 par Tim Berners-Lee, il structure le contenu des pages web et reste la pierre angulaire du développement web moderne, avec plus de 1,5 milliard de sites web l'utilisant.

Saviez-vous ? HTML5, la dernière version majeure, a introduit des éléments sémantiques comme <header>, <footer>, et <article> qui améliorent à la fois la structure et l'accessibilité des sites web.

🛠️ Les fondamentaux de HTML

Structure de base : 
<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <title>Titre de la page</title>
</head>
<body>
  <!-- Contenu visible -->
</body>
</html>
Balises essentielles :
  • <h1> à <h6> : Titres (heading)
  • <p> : Paragraphes
  • <a> : Liens hypertextes
  • <img> : Images
  • <ul>, <ol>, <li> : Listes
  • <div> : Conteneur générique

🎯 HTML5 : Les nouveautés

Éléments sémantiques :
<header> <nav> <section> <article> <footer> <figure>
Multimédia natif : 
<video width="400" controls>
  <source src="video.mp4" type="video/mp4">
  Votre navigateur ne supporte pas les vidéos HTML5.
</video>

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
</audio>

📊 Comparaison des versions HTML

Version Année Principales nouveautés
HTML 2.0 1995 Première version standardisée
HTML 4.01 1999 CSS, frames, scripting
XHTML 1.0 2000 Syntaxe XML stricte
HTML5 2014 Sémantique, multimédia, APIs
Bonne pratique : Validez toujours votre code HTML avec le validateur du W3C (validator.w3.org) pour détecter les erreurs et améliorer la compatibilité.
